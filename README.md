# YC News

A really simple hackernews clone, built with vue js.
Toggle between light and dark theme by clicking the Y-icon.

## Project setup
```

```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
